# Data Sources And Licensing

The repository ships one synthetic replay asset:

- `data/synthetic/canned_replay/demo_conv_bpsk.iq`
- `data/synthetic/canned_replay/demo_conv_bpsk.json`

These files are generated by `scripts/generate_synthetic_iq.py`. They do not
contain captured mission RF, live satellite traffic, or operator-derived data.
The payload is a fixed demo string and the waveform is fully synthetic.

No claim is made that the canned replay matches a specific satellite mission or
air interface.
