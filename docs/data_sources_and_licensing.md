# Data Sources And Licensing

The repository ships two synthetic replay fixtures:

- `data/synthetic/canned_replay/demo_conv_bpsk.iq`
- `data/synthetic/canned_replay/demo_conv_bpsk.json`
- `data/synthetic/canned_replay/demo_conv_bpsk_impaired.iq`
- `data/synthetic/canned_replay/demo_conv_bpsk_impaired.json`

These files are generated by `scripts/generate_synthetic_iq.py`. They do not
contain captured mission RF, live satellite traffic, operator-derived data, or
mission-specific waveform captures.

The payload is a fixed demo string and the waveform is fully synthetic. The
impaired fixture exists to exercise trust-monitoring behavior on a weaker but
still decodable replay.

No claim is made that either fixture matches a specific satellite mission, air
interface, or operational channel model.
